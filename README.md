# SIN3D-Bot

## Description

Discord bot which generates custom links for SIN3D-app.

## Requirements

Set your own configutation:
```
cp .env.example .env
cp config.example.json config.json
```

## How to install ?

```bash
pip install -r requirements.txt
```

## How to launch ?

Just launch the bot:
```bash
python sin3d.py
```
